

Requirements:
python version 3.0 or above (3.4 was used for the purpose of our project)
beautifulsoup 4.0
Libraries:
urllib2
regular expression
csv (to write a csv file of the dataset)


How to use the code:
Run the fineart.py file first it will generate a txt file containing the url of the images
Next run the finear_reading.py it will generate the csv file by taking the previously generated url
txt file as input.
